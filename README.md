## frip
- [프립(Frip)](https://www.frip.co.kr/) 사이트
- 소개: 프립(Frip)은 소셜 액티비티 플랫폼입니다. 사람들을 모아 등산, 윈드서핑, 클라이밍, 베이킹, 소셜 모임, 여행 등의 액티비티를 함께 즐기고 소통하며 더 행복한 삶을 살 수 있게 도와주는 서비스 플랫폼입니다.

## 팀원
